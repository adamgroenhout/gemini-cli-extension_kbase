# Gemini CLI Extension Template
This repo contains the core files and content (minimum requirements) used to create a new Gemini CLI extension.
All content comes from: https://github.com/google-gemini/gemini-cli/blob/main/docs/extensions/getting-started-extensions.md

## Guidance for using this repo
Use as a reference to ensure that all required files are created and content is correctly formatted. 

### How to install extension
gemini extensions install https://github.com/adamgroenhout/gemini-cli-extensions/{extension-name}
